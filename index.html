How to Play?

Select Any Game

Currently the available games are:

  • Marvel Game
  • DC Game
  • Dragon Ball Game

Select Characters

You will be given a selection of characters, first select the character your character then select the opponent character.

Attacking

You will have a few different moves/ attacks based on the game. Select one of them & wait for the enemy to attack your character. And this is to be repeated until either your character or the opponent runs out of HP(hit point).
